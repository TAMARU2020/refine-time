# 💈 Barber Booking System
A modern appointment booking system designed for classic men's barbershops.
The application allows barbers to manage appointments in a simple, clear, and efficient way.

## 🚀 Live Demo

**URL**: 🔗 https://barber-booking-gamma.vercel.app

## 🎯 Project Purpose

This system was built for men’s barbershops that want to:

- Manage appointments without overlaps
- View daily workload clearly
- Allow quick and easy appointment booking
- Replace manual scheduling with a digital solution
- Provide a clean and user-friendly interface

## ✨ Key Features

- Appointment booking with success message and animation
- Confirmation sound upon booking
- Search appointment by phone number
- Daily workload view
- “Copy Details” button
- Admin login with password protection
- Custom UI theme

## 🛠 Technologies Used

- React
- Vite
- TypeScript
- Tailwind CSS
- shadcn-ui

Deployment & Version Control:

- GitHub – source code management
- Vercel – live hosting

The project was developed using an AI-assisted development approach, leveraging an LLM-based tool to accelerate the coding process.
